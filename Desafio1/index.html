O que é um HTML

(Voltar ao início)

HTML (HyperText Markup Language) é a linguagem padrão utilizada para criar páginas na web. Ele funciona como a estrutura básica de um site,
organizando e apresentando o conteúdo por meio de elementos chamados "tags". O termo "HyperText" refere-se à capacidade do HTML de conectar
páginas através de links, permitindo a navegação na internet. Já "Markup Language" significa que ele é baseado em marcações, ou seja, usa tags para
definir a estrutura e a apresentação do conteúdo.
Um documento HTML é interpretado pelos navegadores, que exibem o conteúdo de acordo com a hierarquia definida pelo código, podendo incluir
textos, imagens, vídeos, links, tabelas e outros elementos essenciais para a construção de páginas web.

O que são Tags

(Voltar ao início)

As tags são os elementos fundamentais do HTML e servem para estruturar e organizar o conteúdo de uma página web. Elas são representadas por
palavras-chave entre colchetes angulares (< >) e geralmente aparecem em pares, contendo uma tag de abertura(<nomeDaTag>) e uma de fechamento(</nomeDaTag>).

Usadas para definir títulos e subtítulos, indo do mais importante (<h1>) ao menos importante (<h6>).

Tag de Parágrafo <p>

(Voltar ao início)

A tag <p> é usada para definir parágrafos em um documento HTML. Ela ajuda a organizar o texto de forma estruturada, separando blocos de conteúdo.
Os navegadores aplicam automaticamente um espaçamento antes e depois de cada parágrafo, tornando o texto mais legível. Os parágrafos são essenciais
para criar textos bem organizados e fáceis de ler, contribuindo para uma melhor experiência do usuário. 


            
Este é um exemplo de um parágrafo em HTML. Ele contém um bloco de texto organizado e formatado automaticamente pelo navegador.

            -------------------------------------------------------------------------------------------------------------------------------------------------
            <p>Este é um exemplo de um parágrafo em HTML. Ele contém um bloco de texto organizado e formatado automaticamente pelo navegador.</p>
            -------------------------------------------------------------------------------------------------------------------------------------------------

Tag para Destacar Texto <mark>

(Voltar ao início)

A tag <mark> é usada para destacar partes específicas do texto, geralmente aplicando um fundo amarelo por padrão. Essa marcação é útil para enfatizar
palavras-chave, termos importantes ou trechos relevantes dentro de um texto. de identificar visualmente.

            
O HTML é a linguagem base para a criação de páginas web. A tag HTML é essencial para o desenvolvimento de qualquer site.

            -------------------------------------------------------------------------------------------------------------------------------------------------
            <p>O HTML é a linguagem base para a criação de páginas web. A tag <mark>HTML</mark> é essencial para o desenvolvimento de qualquer site.</p>
            -------------------------------------------------------------------------------------------------------------------------------------------------

Tag de Texto Pequeno <small>

(Voltar ao início)

A tag <small> é usada para reduzir o tamanho do texto em relação ao restante do conteúdo da página. Tradicionalmente, essa tag era utilizada para notas de rodapé,
avisos legais e outras informações secundárias.

                
Este texto está menor

                --------------------------------------------
                <p><small>Este texto está menor</small></p>
                --------------------------------------------

Tag de Texto em Itálico <i>

(Voltar ao início)

A tag <i> é usada para exibir o texto em itálico, geralmente para destacar palavras estrangeiras, pensamentos, títulos de obras ou expressões técnicas dentro de um parágrafo.

                
Ela pensou: Será que isso realmente vai funcionar?

                ----------------------------------------------------------------
                <p>Ela pensou: <i>Será que isso realmente vai funcionar?</i></p>
                ----------------------------------------------------------------

Tag de Texto Sublinhado <u>

(Voltar ao início)

A tag <u> é usada para sublinhar textos, destacando palavras ou frases dentro de um conteúdo. Antigamente, essa tag era amplamente utilizada para enfatizar palavras, mas
com o avanço do CSS, seu uso se tornou menos comum, sendo recomendada apenas em casos específicos, como indicar erros ortográficos ou destacar termos importantes.

                
O professor destacou a palavra importante na lousa.

                -------------------------------------------------------------------
                <p>O professor destacou a palavra <u>importante </u> na lousa.</p>
                -------------------------------------------------------------------

Tag de Texto em Negrito <strong>

(Voltar ao início)

A tag <strong> é usada para destacar uma parte do texto em negrito, indicando que aquele trecho possui importância semântica dentro do conteúdo.
Diferente da tag <b>, que apenas altera a aparência do texto para negrito sem significado adicional, <strong> é interpretada pelos navegadores e
mecanismos de busca como uma ênfase relevante.

                
É fundamental fazer um backup dos seus arquivos regularmente.

                ------------------------------------------------------------------------------------
                <p>É <strong>fundamental<strong> fazer um backup dos seus arquivos regularmente.</p>
                ------------------------------------------------------------------------------------

Tag de Lista Ordenada <ol>

(Voltar ao início)

com a tag <li>, e a numeração padrão segue uma ordem crescente (1, 2, 3...), mas pode ser personalizada com CSS ou pelo atributo type.

Tag de Lista Não Ordenada <ul>

(Voltar ao início)

A tag <ul> é usada para criar listas não ordenadas, onde os itens são exibidos com marcadores (pontos, círculos ou quadrados) em vez de números.
Cada item da lista é definido com a tag <il>. Esse tipo de lista é útil quando a ordem dos itens não é importante, como em listas de compras, características
de um produto ou tópicos de um conteúdo.

Tag de Item de Lista <li>

(Voltar ao início)

A tag <li> (List Item) é usada para definir os itens dentro de listas ordenadas (<ol>) e não ordenadas (<ul>). Cada <li> representa um item dentro da estrutura
da lista, sendo exibido com um número (em <ol>) ou um marcador (em <ul>), como mostrado no exemplos anteriores.

Tag de Link (Âncora) <a>

(Voltar ao início)

A tag <a> é usada para criar hiperlinks, permitindo que os usuários naveguem entre páginas da web, documentos, seções dentro da mesma
página ou até mesmo façam o download de arquivos. O atributo principal da tag <a> é href, que define o destino do link. Ele pode apontar para: 

                
Visite o Google para fazer suas pesquisas.

                -----------------------------------------------------------------------------------------------------
                <p>Visite o <a href="https://www.google.com" target="_blank">Google<a> para fazer suas pesquisas.<p>
                -----------------------------------------------------------------------------------------------------

(Voltar ao início)

Tag Citação em Bloco <blockquote>

(Voltar ao início)

A tag <blockquote> é usada para definir citações em bloco, geralmente empregada para destacar trechos de texto citados de outras fontes.
Os navegadores costumam aplicar um recuo ao conteúdo dentro dessa tag, indicando visualmente que se trata de uma citação.

            

    "No meio da confusão, encontre a simplicidade. A partir da discórdia, encontre a harmonia. 
     No meio da dificuldade reside a oportunidade." – WHEELER, John Archibald
            

        --------------------------------------------------------------------------------------------------------    
        <blockquote>"No meio da confusão, encontre a simplicidade. A partir da discórdia, encontre a harmonia. 
           No meio da dificuldade reside a oportunidade." – WHEELER, John Archibald</blockquote>
        --------------------------------------------------------------------------------------------------------

Tag de Estilização de Texto (Obsoleta) <font>

(Voltar ao início)

A tag <font> era usada para modificar o estilo do texto, permitindo alterar a cor, tamanho e tipo de fonte diretamente no HTML.
No entanto, essa tag foi descontinuada no HTML5, pois a estilização do texto agora deve ser feita com CSS.

                
Este texto está em vermelho, tamanho 5 e fonte Arial.


                <p><font color="red" size="5" face="Arial">Este texto está em vermelho, tamanho 5 e fonte Arial.</font></p>

Tag de Texto Deletado (Riscado) <del>

(Voltar ao início)

A tag <del> é usada para indicar que um trecho de texto foi removido ou modificado em um documento. O conteúdo dentro dessa
tag geralmente aparece riscado, indicando que foi eliminado.

                
O preço original era R$ 100,00, agora está por apenas R$ 80,00!


                <p> preço original era <del>R$ 100,00</del>, agora está por apenas R$ 80,00!</p>

Tag de Abreviação <abbr>

(Voltar ao início)

A tag <abbr> é usada para definir abreviações e siglas, ajudando a fornecer uma explicação sobre o significado de uma palavra ou termo.
Quando o usuário passa o mouse sobre o texto, o significado aparece em forma de tooltip (dica de ferramenta).

                
O HTML é a linguagem usada para estruturar páginas da web.

                <p> O <abbr title="HyperText Markup Language">HTML</abbr> é a linguagem usada para estruturar páginas da web.</p>
